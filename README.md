# Feastopia 🍽️

A modern food ordering web app built with **React**, **Express**, and **MongoDB**.

---

## 🚀 Features

- 🔐 User and Admin Login
- 🛍️ Food Menu with Cart
- 💳 Checkout with UPI / Cash on Delivery
- 📦 Order Tracking
- 📢 Admin Order Management with Notifications
- 🖼️ Food Image Upload
- 📈 Dashboard for Admin

---

## 🛠️ Tech Stack

- **Frontend:** React, React Router, CSS, Font Awesome
- **Backend:** Express.js, Node.js, MongoDB
- **Others:** JWT Auth, LocalStorage, morgan, dotenv

---

## 📦 Installation

```bash
# Clone the repo
git clone https://github.com/your-username/feastopia.git
cd feastopia

# Install dependencies
npm install

# Go to backend folder and install
cd backend
npm install

# Create .env file
touch .env
